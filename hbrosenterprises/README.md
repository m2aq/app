# 🌾 HBros Enterprises - Ranch Field Activity Monitor

Real-time work progress tracker for ranch fields with satellite map integration.

## Features

- 🛰️ Satellite view with field polygons from KML
- 📊 Progress tracking for 8 activities per field
- 🗺️ Interactive Leaflet map with exact field shapes
- ➕ Add/update fields dynamically
- 📱 Responsive design with Tailwind CSS

## Activities Tracked

- Wire Removal
- Hose Removal
- Cutting Post Wire
- Post Removal
- Plant Removal
- Cleaning
- Ready to Burn
- Ready to Collect

## How to Use

1. Open `index.html` in a browser
2. View fields with their progress on the satellite map
3. Click "Update Progress" to modify activities
4. Add new fields with coordinates

## Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR-USERNAME/ranch-monitor.git
git branch -M main
git push -u origin main
```

Then enable GitHub Pages in repository Settings.
